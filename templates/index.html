How DrowsyGuard Works

Backend (Python): Flask server runs MediaPipe face detection + EAR + PERCLOS + Head Pose + CNN + ANN every frame. Results are sent to this page via HTTP polling.

CNN: MobileNetV2 classifies each eye region (64×64 image) as open/closed. Gives a probability 0–1.

ANN: Takes 6 features (EAR, PERCLOS, blink rate, pitch, yaw, MAR) → outputs a unified fatigue score 0–100.

Alert levels: Determined by the ANN score with PERCLOS + consecutive-closed-frame overrides.

To use: Run python app.py in your terminal, then open http://localhost:5000
